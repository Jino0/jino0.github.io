打开终端

输入命令brew install hugo,此命令可直接安装hugo(当然，前提是将brew安装好)，hugo version可查看是否已经安装好。

输入命令hugo new site myblog,最后一个单词为在当前目录下生成文件夹名，也是你的网站的大本营，后续大部分的操作都在这个文件夹下进行，此目录在下文中我统称为根目录。

可在hugo官网主题进行下载主题,可以使用git命令对主题进行克隆，注意，下载的主题要放在博客根目录的themes目录下。

在根目录下执行命令hugo server -t hugo-tranquilpeak-theme –buildDrafts,其中hugo-tranquilpeak-theme为你下载的主题名称,如果没问题就会提示下面信息，此时在浏览器中输入:localhost:1313,如果能正常显示，说明在本地服务器运行已经没问题。
